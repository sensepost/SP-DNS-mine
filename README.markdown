# 4. Description
SP-DNS-mine.pl uses Google to extract sub domains and DNS names for a given domain. This PERL script features in J0hnny Long's book "Google Hacking for Penetration Testers".

# 5. Usage
> perl sp-dns-mine.pl cnn.com
